## Controls
RadioButtons - mute and unmute instruments and change instrument patterns.
Play & Stop - Plays and stops the music respectively.
Save & Load - Saves and loads the current button layout to the browsers local storage as a json
Volume Slider - Affects the gain of each instrument resulting the overall volume
Reverb Slider - Applies reverb to everything using the .room() attribute
CPM - Changes the tempo of the song
Slider Toggles - applies filters to song including a highpass, lowpass, and Overdrive

## Quirks
When you first start the server the instruments will say "undefined" infront but it still works!
Just need to click any radio button and it will either be blank ("") or muted ("_").
